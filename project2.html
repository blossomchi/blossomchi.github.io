Smart Bin System

Waterfall-model design, sensor integration, and automated waste sorting prototype built using Arduino.

Project Overview

The Smart Bin System is an automated waste-sorting solution that uses sensors and Arduino-based control logic to identify, detect, and sort different waste types. Built using the Waterfall model, the project demonstrates every phase of the software engineering lifecycle.

Development Approach (Waterfall Model)

  • Requirements Analysis: Define categories, sensor type, sorting logic, and system goals.
  • System Design: Create electronics schematics and mechanical layout.
  • Implementation: Develop Arduino logic and assemble hardware.
  • Testing: Validate sensor accuracy and servo movement.
  • Deployment: Install and run in a controlled environment.
  • Maintenance: Improve detection thresholds and tune alignment.

Hardware Used

  • Arduino UNO
  • Ultrasonic Sensor
  • IR Sensor
  • Servo Motor
  • LED Indicators
  • Plastic Container

Sample Arduino Code Snippet

// Smart Bin System – Simplified Code
#include 

Servo flap;
int irPin = 2;

void setup() {
  pinMode(irPin, INPUT);
  flap.attach(9);
}

void loop() {
  int detected = digitalRead(irPin);

  if (detected == LOW) {
    flap.write(90);   // Open
    delay(1500);
    flap.write(0);    // Close
  }
}
