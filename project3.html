House Price Prediction Project

Linear regression model + Streamlit app using Bengaluru dataset for interactive price estimation.

Project Overview

This Machine Learning project predicts house prices using the Bengaluru House Dataset. Users can enter values such as square footage, number of bedrooms, and location through a Streamlit interface, and the system returns an estimated price.

Key Features

  • Linear Regression model for predicting house prices
  • Cleaned and preprocessed Bengaluru dataset
  • Interactive Streamlit interface
  • One-hot encoded location categories
  • Real-time prediction output with confidence awareness

Technologies Used

Python Pandas Scikit-learn Streamlit NumPy

Model Building Steps

  • Data Cleaning: Remove duplicates, drop outliers, fill missing values.
  • Feature Engineering: One-hot encoding for locations, normalize values.
  • Model Training: Linear Regression.
  • Streamlit UI: User inputs → Model → Predicted price.

Sample Code Snippet

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

df = pd.read_csv("Bengaluru_House_Data.csv")
df = pd.get_dummies(df, drop_first=True)

X = df.drop("price", axis=1)
y = df["price"]

model = LinearRegression()
model.fit(X, y)

def predict(features):
    return model.predict([features])[0]

Project Outcome

The House Price Predictor helps users estimate prices instantly. It serves as a practical introduction to regression modeling, data preprocessing, and interactive machine learning app deployment.
